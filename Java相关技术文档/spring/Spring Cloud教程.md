# 微服务架构与Spring Cloud

微服务架构是一种架构模式，它提倡将单一应用程序划分成一组小的服务，服务之间互相协调、互相配合，为用户提供最终价值。

每个服务可以独立部署。

spring cloud是一系列服务的集合。
